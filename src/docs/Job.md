# NotehubJs.Job

## Properties

| Name           | Type                 | Description                               | Notes      |
| -------------- | -------------------- | ----------------------------------------- | ---------- |
| **created**    | **Number**           | Unix timestamp when job was created       |
| **createdBy**  | **String**           | User who created the job                  |
| **definition** | **{String: Object}** | Full job definition (only in detail view) | [optional] |
| **jobUid**     | **String**           | Unique identifier for the job             |
| **name**       | **String**           | Human-readable job name                   |
